1.URI编码方法

Global对象的encodeURI()和encodeURIComponent()

encodeURI()主要用于对整个URI,而encodeURIComponent主要对URI中的某一段进行编码。它们的主要区别在于，encodeURI()不会对特殊的字符进行编码，例如冒号、正斜杠、问号和井字号；而encodeURIComponent()则会对他们发现的任何非标准字符进行编码。

 var uri = 'https://juejin.im/  post/5c342e0fe51d45524975d9c1';
 encodeURI(uri);   //"https://juejin.im/%20post/5c342e0fe51d45524975d9c1"
 encodeURIComponent(uri);  //"https%3A%2F%2Fjuejin.im%2F%20post%2F5c342e0fe51d45524975d9c1"

使用encodeURI()编码后的结果是除了空格之外的其他字符原封不动，只有空格被替换为%20，而encodeURIComponent()则会替换所有的非字母的数字字符。这也是可以对整个URI使用encodeURI(),而只能对附加在现有的URI后面的字符串使用encodeURIComponent()的原因所在。

与encodeURI()和encodeURIComponent相对应的方法分别是decodeURI()和decodeURIComponent().作用与encodeURI()和encodeURIComponent相反。

 var uri = 'https://juejin.im/%20post/5c342e0fe51d45524975d9c1';
 decodeURI(uri);   //https://juejin.im/  post/5c342e0fe51d45524975d9c1"
 decodeURIComponent(uri);   //'https://juejin.im/  post/5c342e0fe51d45524975d9c1'

min()和max()方法

用于确定一组数组中最大值和最小值，这两个方法都可以接受多个数值参数。

var max = Math.max([23,34,26,56,71])
alert(max) //71
var min =  Math.min([23,34,26,56,71])
alert(min) //23

也可以借用Math的max()和min()方法

var value = [23,34,26,56,71]
var max = Math.max.apply(Match,value)

这个技巧的关键是把Math对象作为apply()的第一个参数，从而正确的设置this值。然后，可以将任何数组做为第二个参数

舍入方法

  • Math.ceil()向上舍入 （Math.ceil(2.1) 为3）
  • Math.floor()向下舍入 （Math.floor(2.1) 为2）
  • Math.round()标准舍入（即四舍五入）

random()方法

Math.random()返回任意一个大于或等于0小于1的数（0 <= Math.random() < 1）

  var num = Math.floor(Math.random() *10 + 1); //返回一个0~10的随机数
